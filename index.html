Language rules

Third-person self
"my crew / me"
→ "Rocky crew" (no 's)
No articles
"a / an / the"
→ dropped
No auxiliaries
"will / would / shall"
→ dropped (keeps "can")
"Is" at start only
"it is good / Grace is"
→ "Is good" / "Grace"
Direct negation
"I don't know"
→ "Rocky not know"
Question tag
"Are you okay?"
→ "…, question?"
Statement tag
Emphatic declarations
→ "…, statement."
Repetition = intensity
"very bad / amazing"
→ "bad, bad, bad"
Short sentences
Long compound phrases
→ split on and/but/because
Drop infinitive "to"
"need to know"
→ "need know"
Present tense
"I went / she gave"
→ "Rocky go / she give"
No filler words
"just / actually / really"
→ dropped
